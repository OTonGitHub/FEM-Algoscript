### Setup History

#### Git

`git init`

`git branch -M main`

`git add -A .`

`git commit -am "init commit"`

`npm init -y`

#### Node

`npm install typescript --save-dev`

`npm install --save-dev tsx`

`npx gitignore node typescript vscode`

#### Git Again

`git rm -r --cached .` # untrack from before gitignore

`git add -A .`

`git commit -am "retrack"`
